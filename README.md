# DynamicBoneCopy
DynamicBoneの設定内容の出力、コピー、ペーストができます。


DynamicBone上で右クリックすることで、メニューが表示されます。

エクスポートファイルは「Assets\ExpansionTools\DynamicBoneCopy\Datas」配下に出力されます。
